# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by creating an issue or contacting the maintainer directly.

Please do not create public issues for security vulnerabilities until they have been addressed.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Security Best Practices

- Keep dependencies up to date
- Review code changes before merging
- Follow Go security best practices
